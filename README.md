# Personal Portfolio Website

This is a modern personal portfolio website developed to showcase my skills, projects, and academic background. The website is designed with a clean user interface, smooth animations, and full responsiveness across different devices.

## 🚀 Features

* Responsive multi-section layout (Home, About, Skills, Projects, Contact)
* Modern dark-themed UI with gradient accents
* Sticky navigation bar with scroll effects
* Mobile-friendly hamburger menu
* Smooth scrolling and active link highlighting
* Scroll-based animations using Intersection Observer
* Project showcase with GitHub and live demo links

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* DOM Manipulation
